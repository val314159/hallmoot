# Sandbox Project Examples

## python_calc
Simple Python calculator with add/subtract.
- LLM can edit `calc.py` to add operations.
- Use `run_make test` to add pytest tests.
- Use `run_make run` to execute.

## c_hello
Minimal C program.
- LLM can modify `hello.c` and rebuild with `run_make`.
- Add new targets to Makefile via `write_file`.

Drop any of these into `sandbox/` and start developing with `hallmoot`.
